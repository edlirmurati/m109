Cloud Computing

Cloud Computing ermöglicht es, Computerressourcen wie Rechenleistung und Speicherplatz über das Internet bereitzustellen und zu nutzen, ohne physische Hardware vor Ort zu benötigen.

css

Was ist eine Cloud an sich

Cloud-Plattformen können viele Dienste wie virtuelle Maschinen, Speicher, Datenbanken, Netzwerk- und Anwendungsunterstützung bereitstellen, die von Benutzern genutzt werden können. Es ist einfach gesagt ein Konzept für das Bereitstellen und Nutzen von Computerressourcen über das Internet.
